

| 功能编号  | 功能名称   | 描述                                                      | 功能区块 | 优先级 |
| ----- | ------ | ------------------------------------------------------- | ---- | --- |
| FR-01 | 音频输入   | 支持本地文件（MP3/WAV/FLAC/M4A）、YouTube链接、Bilibili链接           | 核心功能 | P0  |
| FR-02 | 音轨分离   | 分离为6轨（人声/鼓/贝斯/钢琴/吉他/其他）支持进度显示，至少一个模型BS-RoFormer         | 核心功能 | P0  |
| FR-03 | 音轨选择   | 从分离结果中选择目标音轨进行分析                                        | 核心功能 | P0  |
| FR-04 | 和弦分析   | 301类和弦识别，输出和弦级数与时间轴                                     | 核心功能 | P0  |
| FR-05 | 播放控制   | 播放/暂停、跳转、调速（0.5x~2x）、循环A-B段                             | 核心功能 | P0  |
| FR-06 | 可视化    | 波形、节拍标记、和弦标签、节奏型图谱                                      | 核心功能 | P0  |
| FR-07 | 音乐车间   | 新建/加载/保存/切换车间，一个车间 = 一段音频 + 四个Tab的状态                    | UI   | P1  |
| FR-08 | 分离模型选择 | 显示本地可用模型、自动推荐最优模型、API Key接入                             | 核心功能 | P1  |
| FR-09 | 节奏分析   | BPM、拍号、节拍位置、常见节奏型识别                                     | 核心功能 | P1  |
| FR-10 | 轨道控制   | 各轨道独立静音/独奏                                              | 核心功能 | P1  |
| FR-11 | MIDI导出 | 将分离轨导出为标准MIDI文件                                         | 核心功能 | P1  |
| FR-12 | 个性化插件  | 利用核心分析得出的结果，可以选择性地进行更加高级细致的分析处理，通过PluginManager调用各种拓展插件 | 核心功能 | P2  |
